# Basic API Project

This project shows how to use a basic API in Node.js using a .env file for secrets.

## Files Included:
- `index.js` – Main code
- `notes.pdf` – Notes explaining the concept
- `.env` – Secret keys (NOT uploaded)

## How to Use:
1. Clone the repo
2. Create your own `.env` file
3. Run using `node index.js`
